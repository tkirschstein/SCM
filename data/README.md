# Data Files

This folder contains datasets used in exercises and case studies.

| File | Description | Used in |
|------|-------------|---------|
| `battery_plants.csv` | Automotive plants with battery demand (MWh/yr) | Case Study 1 |
| `battery_warehouses.csv` | Potential warehouse locations with fixed costs | Case Study 1 |
| `transport_distances.csv` | Pairwise distances between warehouses and plants | Case Study 1 |
| `demand_fashion.csv` | Synthetic historical demand data for fashion retailer | Case Study 2 |
| `pear_diskdrive.csv` | Transportation problem — Pear Disk Drive | Exercise 3 |
| `rheingau_locations.csv` | Location data for Steiner-Weber exercise | Exercise 2 |
